💳 Conta Bancária em Java

Projeto simples desenvolvido em Java com o objetivo de praticar Programação Orientada a Objetos (POO).

Este projeto simula um sistema bancário básico executado no console, permitindo a criação de uma conta e a realização de operações bancárias simples.


📚 Conceitos praticados

Programação Orientada a Objetos (POO)

Classes e Objetos

Encapsulamento

Métodos

Estruturas condicionais (if/else)

Estruturas de repetição (while)

Validação de regras de negócio


⚙️ Funcionalidades

Criar uma conta bancária

Depositar dinheiro

Sacar dinheiro

Consultar saldo

Menu interativo no console

Validação de saldo para saque


▶️ Como executar

Clone o repositório
git clone https://github.com/Jotebs/Conta-Bancaria-em-Java.git
Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou VS Code)
Execute o arquivo:
Main.java


🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar os fundamentos de Java e Programação Orientada a Objetos, servindo como exercício de aprendizado durante os estudos da linguagem.
